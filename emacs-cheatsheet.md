# Emacs Cheat Sheet

Comment codeblock 

``` M-x ```

Uncomment codeblock 

```M-x ```

Close all windows except the one the cursor is in 

``` C-x 1```

Set region

``` C-SPC ```

Clear region

``` C-w ```

Copy region

``` M-w ```

To to indent

``` C-x TAB ```

Resize buffer

``` C-u -4 C-x ^ ```
